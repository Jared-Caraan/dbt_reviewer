What is the primary function of a package in a dbt project?

a. To manage database connections more efficiently.\
b. To optimize query performance.\
c. To import and reuse models, macros, and other assets from external sources.\
d. To automate the deployment of dbt projects

Answer: c

##
Which of the following commands is used to install a package in your dbt project?

a. `dbt install`\
b. `dbt deps`\
c. `dbt fetch`\
d. `dbt add-package`

Answer: b
