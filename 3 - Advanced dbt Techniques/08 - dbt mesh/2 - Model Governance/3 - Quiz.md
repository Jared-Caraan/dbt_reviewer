Select all options below that describe benefits of using model contracts with dbt Cloud

a. Model contracts ensure improved data accuracy across projects.\
b. Constraints in model contracts guarantee that you will never see invalid data in the table materialized by your model.\
c. Model contracts ensure a model will never fail to build.\
d. Model contracts increase security measures for database connections.

Answer: a,b

##
What two things will dbt check when it sees that a model has enforced contracts?

a. The existence of all columns listed in the contract.\
b. The correct order of all columns listed in the contract.\
c. The data types of all columns listed in the contract.\
d. The length of all columns in the contract.

Answer: a,c

##
In a dbt build, when does dbt enforce a contract?

a. Before tests run.\
b. Alongside tests, in DAG order.\
c. After tests run.

Answer: a
