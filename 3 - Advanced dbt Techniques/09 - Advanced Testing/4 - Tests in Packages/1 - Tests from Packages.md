## Tests in Packages

**What a test that isn't one of the four native dbt tests?**

You can find most tests you need in packages, so you don't need to write them frpm scratch.

**Must have testing pakcages:** `dbt_utils`, `dbt_expectations`, `audit_helper`

**How do you install a package?**
- Import the package you want into your packages.yml file.
- Add the test to your model.yml file

**Explore all packages on [hub.getdbt.com](https://hub.getdbt.com/)**
